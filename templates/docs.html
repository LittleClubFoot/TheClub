üê≥ Docker Services

üÜò Troubleshooting

Common issues and solutions:

Services Not Starting

  • Check Docker is running: docker ps
  • View logs: make logs
  • Restart services: make docker-stop && make docker-dev

SSL Certificate Issues

  • Ensure domain points to your server
  • Check firewall allows ports 80/443
  • View Caddy logs for certificate errors

Service Connectivity

  • Verify NAS services are accessible from dashboard server
  • Check Caddyfile routing configuration
  • Test direct service URLs
