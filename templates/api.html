Available Endpoints

GET /test

Main test server page with HTMX demo

Response: HTML page

Description: Interactive page demonstrating HTMX functionality with dynamic time updates

GET /test/time

Returns current server time (HTMX endpoint)

Response: HTML fragment with current time

Example: <div class="time-display">Current time: 14:30:25</div>

Usage: Called by HTMX from the main test page

GET /test/api

This API documentation page

Response: HTML page

Description: Complete documentation of all available endpoints

GET /test/docs

General documentation page

Response: HTML page

Description: Project documentation and usage guides

GET /health

System health check endpoint

Response: Plain text "OK"

Status Code: 200

Usage: Monitoring and load balancer health checks

Integration

This Go server is designed to work behind a Caddy reverse proxy:

  • Homer Dashboard: / (port 8080)
  • Go Test Server: /test (port 8082)
  • NAS Services: Various routes (configured in Caddyfile)

HTMX Integration

The test server demonstrates HTMX capabilities:

  • Dynamic Updates: Content updates without page reloads
  • Server-Side Rendering: HTML fragments returned from server
  • Progressive Enhancement: Works without JavaScript
  • Minimal Footprint: Lightweight client-side library

Example Usage

Test the HTMX functionality:

  1. Visit /test
  2. Click the "Get Current Time" button
  3. Observe the dynamic content update
  4. Check browser network tab to see HTMX request to /test/time
